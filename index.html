How to install Python enviroment?

Python 3 Python 2
  1. Go to the www.python.org
  2. In the menu click Downloads
  3. Scroll down to the 'Looking for a specific release?' table and click on Download button with the newest version of Python 3.
  4. Scroll down to the 'Files' table and in the 'Version' column click Windows x86-64 executable installer or Windows x86 executable installer(depends on system).
  5. Open the downloaded file.
  6. Select 'Add Python 3.7 to PATH' and click 'Cusomize installation'.
  7. Just press 'Next' in the next screen.
  8. In ths window we must specify where will the files will we stored. Click 'Browse' and install it in the folder which is most suitable for you.
  9. Open cmd and type 'python' to check if the package is correctly installed. (To exit the program enter 'exit()')
  10. Optional if you are installing two enviroments:
    • Go to the Python folder and change the name of the python.exe to python3.exe and go to the 'Scripts' and change name of the pip manager from pip.exe to pip3.exe so it wont interfere with your second enviroment command for the python.

Python two enviroments installation troubleshooting

Fatal error in launcher: Unable to create process

  • The pip packages manager for python has tendency to crash when the two enviroments are installed.
  • My solution to the problem was to upgrade the manager so it will overwrite his default values with new, correct ones using:
  • python -m pip install --upgrade pip
  • or just use pip using workaround
  • 'PATH_to_pip'/pip.exe command

'python' is not recognized as an internal or external command, operable program or batch file.

  • When you have installed two enviroments the should be 4 new PATH added to the system.
    1. 'path_to_the_folder'\Python27\Scripts\
    2. 'path_to_the_folder'\Python37\Scripts\
    3. 'path_to_the_folder'\Python37\
    4. 'path_to_the_folder'\Python27\
  • The name of the folder might change depending of the version of the Python.
  • How to check if you have all necessary PATHs?
    1. Open 'Computer' -> right click -> Near the bottom of the menu -> 'Properties'
    2. On the left -> 'Advanced system settings'
    3. Click button 'Environment Variables'
    4. In the table "Environment variables for the user" click row entitled "PATH"
    5. Check if there are required PATHs, if not add them using button on the right -> "New"
  • or just use pip using workaround
    1. 'PATH_to_pip'/pip.exe command
